Pixel Physics

This is a project for NWAPW

Our goal was to make a realtime physics game built around pixels, each pixel is one unit of a substance and there are different substances that behave differently and can interact with each other. Some substances create other ones, like fire creates smoke, and some substances can combine into ones with different properties, like water + sand = wet sand.
